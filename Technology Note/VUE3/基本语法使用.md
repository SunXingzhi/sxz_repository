#  Template Grammar
## vue Attributes 
在vue的模版语法中， 拓展了很多原本HTML标签语法中不带有的功能，比如动态改变DOM。这部分在vue之前是通过js操作DOM元素实现的。不过在vue中可以简单的实现。就是通过`v-bind:`。具体是通过`v-bind:<attributes>=<js_variable_name>`.For example:
```
<template
```
## 事件监听：v-on/@...
我们已经知道的是， vue的模板语法就是html的拓展板， 因此模板语法的功能拓展都是基于html的， 而html元素中的事件处理也不意外。事件中用的比较多的有：鼠标事件（点击clicked，左键，右键等等）。
