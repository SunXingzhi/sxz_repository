# 节区/段.section

汇编更接近底层, 为了可以直接操作内存, 汇编语言定义了一些特定概念.比如段.

| 段                    | 内容                 | 映射到链接脚本的关键字 |
| --------------------- | -------------------- | ---------------------- |
| .text                 | 存放代码指令的位置   | *(.text)               |
| .rodata               | 只读数据()           | *(.rodata)             |
| .data                 | 已初始化全局变量     | *(.data)               |
| .bss                  | 未初始化全局变量     | *(.bss) or COMMON      |
| .vectors/ .isr_vector | 中断向量表(ARM 架构) | 自定义段名             |
|                       |                      |                        |

> Note: 这里说的只读数据,  在C中不一定是加了const就放在rodata区中. 还需要根据const修饰的对象, 以及赋值是否是**<u>立即数</u>**决定.  详情须看C章节中的const 区分.
