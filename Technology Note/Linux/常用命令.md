# 压缩与解压缩

## tar

```
tar [argv] [compress_target_file] [compressed_file]
```

参数如下:

| 参数 | 作用             |
| ---- | ---------------- |
| -c   | 创建新的压缩文件 |
| -C   | 切换到指定目录   |
| -v   | 显示压缩过程日志 |
| -x   | 解压缩文件       |
| -f   | 指定压缩文件     |
|      |                  |

例如如果想要解压缩一个特定位置的文件, 应该是:

```bash
tar -xvf [testfile] -C [target_dic]
```

# 查找文件内容

## grep

命令格式:

```bash
grep [options] pattern(匹配的具体内容,支持正则表达式) [file](可支持多个文件)
```

options 可选.
