# NeoVim配置

## 一站式配置: LazyVIm

前提: 下载好neovim. Ubuntu:`sudo apt install neovim`

官方仓库:[LazyVim/LazyVim: Neovim config for the lazy](https://github.com/LazyVim/LazyVim#)

You can find a starter template for **LazyVim** [here](https://github.com/LazyVim/starter)
