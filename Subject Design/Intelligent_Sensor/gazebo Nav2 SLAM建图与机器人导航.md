# 工作环境


# 作业要求参数
| 作业要求              | 实现方式                               |
| ----------------- | ---------------------------------- |
| 场景 8m×5m          | SDF 中 `<plane> size="8 5"`         |
| 入口 (0.5, 0.5)     | 绿色方块标记，机器人初始位姿                     |
| 出口 (7.5, 4.5)     | 红色方块标记                             |
| 5 个黑色障碍物          | SDF 中 5 个黑色 `<box>`，均匀分布           |
| 机器人 0.5×0.3×0.25m | URDF `<box size="0.5 0.3 0.25">`   |
| 蓝色金属质感            | SDF material blue metallic         |
| 2 驱动轮 + 差速驱动      | `DiffDrive` 插件                     |
| 1 随动轮             | 自由旋转关节                             |
| 激光雷达 /scan        | `GpuLidar` 插件 + `ros_gz_bridge` 桥接 |
| 质量 2kg            | URDF `<mass value="2.0">`          |
| 最大速度 0.5m/s       | `DiffDrive` 参数                     |
