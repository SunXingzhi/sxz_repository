# 核心功能需求

## 统计
### 日统计
每天(24:00)统计出温湿度, CO2 平均值(web http query), 与当前番茄长势判断, 环境是否适合生长;

同时每天隔1h获取一次传感器信息, 以及番茄当前长势
### 周统计(可选)
周统计每天统计番茄的各项判断长势的指标变化, 以及番茄在这周经历的阶段.



## 实时检测
### 温湿度, CO2
根据设定的范围给出提示(超标/正常/偏低).
### 实景图像
web页面单独开出一个窗口来显示实时图像.


# 硬件设计
采用ESP32 + 树莓派作为硬件主控平台,  ESP32主要负责接收上端摄像头数据, 无线方式传输数据到树莓派上.

# 网站设计
## 技术栈选择
前端使用`vue3+vite`构建, 后端使用`maven` + `SpringBoot v3.x`构建, `java`版本:25(LTS)
## 环境搭建
### 前端环境
1. 安装Nodejs(不是太老的版本都可以);
2. 确保环境变量已经添加, 以及vscode安装;
3. vscode打开终端确定`npm -v`, `node -v`查看是否有正确输出(是否提示版本号);
4. 在vscode中创建vue项目. 使用vscode 打开一个文件夹, 作为你的项目文件夹, 在终端输入:`npm create vue@latest`, 然后会提示一些内容, 除了项目名称自定义之外, 其他的都可以选择No(TS支持可选);
5. 创建好项目后, 终端进入同名文件夹, 使用npm run dev, 查看是否创建成功(成功的标志是出现一个网址), 按住`Ctrl`同时鼠标左键点击URL, 浏览器会跳转到vite初始页面, 证明配置成功.
