## 串口绘图器调参指南

### 数据流格式

```
D:1  → 开启数据流
输出：θ,θ̇,x,motor_rpm
0.0120,-0.3450,0.0032,120.5
0.0110,-0.3200,0.0041,118.2
...
```

用 **SerialPlot**（Windows 软件）或 VS Code 插件，设置 4 条曲线，分隔符逗号。

---

### 第一步：只调角速度环（最内环）

```
F:0,0,0          ← 关角度环
E:0,0,0          ← 关位移环
G:5.0,2.0,0.5    ← 给角速度环参数
B:1              ← 启动
D:1              ← 开数据流
```

此时看 **θ̇ 曲线**（第 2 条）：

```
θ̇ 理想波形（用手拨摆杆后松手）：
      │
 0.5  │    ╱╲
      │   ╱  ╲
 0.0  │──╱────╲────────── ← 很快回到 0
      │ ╱      ╲
-0.5  │╱        ╲
      └────────────────── time

θ̇ 振荡不收敛（Kp 太大）：
      │  ╱╲  ╱╲  ╱╲
 0.0  │─╱──╲╱──╲╱──╲──── 
      │╱
      └────────────────── time
      → 减 G 的 Kp

θ̇ 收敛太慢（Kp 太小）：
      │
 0.3  │    ╱╲
      │   ╱  ╲
 0.0  │──╱──────╲──────── ← 很久才回 0
      │╱          ╲
      └────────────────── time
      → 加 G 的 Kp

θ̇ 有毛刺（Kd 太大）：
      │ ╱╲╱╲╱╲
 0.0  │╱────────╲──────── ← 抖动
      └────────────────── time
      → 减 G 的 Kd
```

**调好的标志**：拨摆杆后，θ̇ 在 **0.1s 内** 回到 0，无振荡，无毛刺。

---

### 第二步：加角度环（中间环）

```
F:3.0,3.0,0.3    ← 给角度环参数
B:1
D:1
```

此时看 **θ 曲线**（第 1 条）：

```
θ 理想波形（用手拨 10° 后松手）：
      │
 0.17 │    ╱╲
      │   ╱  ╲
 0.0  │──╱────╲────────── ← 回到 0，1~2 次振荡
      │╱      ╲
-0.17 │        ╲
      └────────────────── time

θ 等幅振荡（Kp 太大）：
      │ ╱╲  ╱╲  ╱╲  ╱╲
 0.0  │╱──╲╱──╲╱──╲╱──╲ ← 一直晃
      └────────────────── time
      → 减 F 的 Kp

θ 收敛太慢（Kd 太小）：
      │    ╱╲
      │   ╱  ╲  ╱╲
 0.0  │──╱────╲╱────╲─── ← 好几次才停
      └────────────────── time
      → 加 F 的 Kd

θ 有静差（Ki 太小）：
      │
 0.05 │────────────────── ← 停在 0.05 rad 不回 0
 0.0  │
      └────────────────── time
      → 加 F 的 Ki

θ 超调过大（Kp 太大或 Kd 太小）：
      │
 0.2  │    ╱╲
      │   ╱  ╲
 0.0  │──╱────╲──────────
      │╱        ╲
-0.15 │          ╲╱       ← 反向超调
      └────────────────── time
      → 减 F 的 Kp 或加 F 的 Kd
```

**调好的标志**：拨摆杆 ±10° 后，θ 在 **0.5s 内** 回到 0，1~2 次振荡收敛。

---

### 第三步：加位移环（最外环）

```
E:2.0,0.1,1.0    ← 给位移环参数
B:1
D:1
```

此时看 **x 曲线**（第 3 条）：

```
x 理想波形（推平台 5cm 后松手）：
      │
 0.05 │    ╱╲
      │   ╱  ╲
 0.0  │──╱────╲────────── ← 回到 0，摆杆不倒
      └────────────────── time

x 回不去（Kp 太小）：
      │
 0.03 │────────────────── ← 停在 0.03m
 0.0  │
      └────────────────── time
      → 加 E 的 Kp

x 回去了但摆杆倒了（Kp 太大）：
      │    ╱╲
 0.0  │───╱──╲─────────── ← x 回来了
      │  ╱    ╲
      │ ╱      ╲          ← 但 θ 发散了（看 θ 曲线）
      └────────────────── time
      → 减 E 的 Kp
```

**调好的标志**：推平台 ±5cm 后，x 在 **2~3s 内** 回到 0，全程摆杆不倒。

---
### 速查表

| 波形特征  | 原因       | 调什么        |
| ----- | -------- | ---------- |
| 振荡不收敛 | 比例增益过大   | **减 Kp**   |
| 收敛太慢  | 比例增益过小   | **加 Kp**   |
| 有静差   | 积分不够     | **加 Ki**   |
| 超调大   | 微分不够     | **加 Kd**   |
| 高频毛刺  | 微分过大或噪声  | **减 Kd**   |
| 电机抖动  | 内环 Kp 太大 | **减内环 Kp** |
| 平台一直跑 | v_ref 漂移 | 检查泄漏积分系数   |
| 摆杆倒了  | 外环 Kp 太大 | **减外环 Kp** |
|       |          |            |
|       |          |            |

---
### 调参口诀

> **内环快（Kp 大，响应 < 50ms）**
> **中环稳（Kp 中，超调 < 20%）**
> **外环慢（Kp 小，响应 < 3s）**
> **先 P 后 D 再 I，一个一个加**
