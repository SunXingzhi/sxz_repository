学习了const关键字, 并且理解了以下两种不同的const 位置, 所代表的实际含义:

| example       | const 实际含义                                                        |
| ------------- | ----------------------------------------------------------------- |
| const int a;  | 当变量是普通变量时, 放在前面放在后面都是等价的                                          |
| int const a;  | 和上面意义相同                                                           |
| const int* a; | 代表`a`是一个指向整数常量的指针, 因此指针本身可以修改, 不过要保证修改后的指针仍然指向整数常量;而指针指向的数据无法被更改; |
| int* const a; | 代表a是指向整数的常量指针, 即指针不可以修改, 但是指向的数据可以被修改;                            |
口诀: 只需要看`const`和谁离得近, 谁就是不可变的.
